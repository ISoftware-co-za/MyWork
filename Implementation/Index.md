# Sprints

A sprint is 20 hours a week. The following story point estimation is used:

| Story Point | Amount of sprint |
| ----------- | ---------------- |
| 1           | 0.03125          |
| 2           | 0.0625           |
| 4           | 0.125            |
| 8           | 0.25             |
| 16          | 0.5              |
